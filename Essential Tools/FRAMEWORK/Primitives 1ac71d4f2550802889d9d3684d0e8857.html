ðŸ”¸VariableReference<T>

Info:

Variable Reference <T> allows developers to create ScriptableObject-based primitive variables, enabling reusable data across multiple logic areas. This implementation of the ScriptableObject Architecture Pattern (SOAP) provides the flexibility to use either the Scriptable Variable or its direct value.

View:

How To Use:

	public BoolVariable boolVariable;
	public BoolReference boolReference;
	
	if (boolReference.Constant)
	{
		boolReference.ConstantValue = true; // your value
	}
	else
	{
		boolReference.Variable = boolVariable; // your reference for your BoolVariable
	}

public class MyType : VariableReference<MyType> {} // Where MyType = your custom Type

ðŸ”¸SecVariable (struct)

Info:

SecVariable is a lightweight primitive designed to obfuscate its memory pointer and store its value in a binary format. This guards sensitive data, such as passwords, coins, and player data, preventing unauthorized viewing or modification.
