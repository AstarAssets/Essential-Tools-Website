ðŸ”¹ Manual Installation

  1. Close the Utility Panel:

    Begin by closing the utility panel in Unity to ensure an unobstructed setup process.

  1. Review Documentation:

    Refer to the Essential Tools documentation to identify the required definitions and dependencies for your package.

  1. Download Dependencies:
    • Use the Unity Package Manager or visit the official website to download the necessary dependencies.
    • For example, if using DOTween, download it and confirm that the DOTween Modules are installed properly.
    • Dependencies can be downloaded and installed by this doc, just check Manual Installation Dependency section in each product.
  1. Configure Project Settings:
    • Navigate to Project Settings > Player > Other Settings within Unity.
    • Locate the section for scripting definitions.
    • For your convenience you can put the folder and definition on the scriptable object Essential Tools Definitions for easy setup.
    • Manually add the required definitions as outlined in the documentation.
  1. Apply Changes:

    Click Apply to save your changes. Once completed, your Essential Tools package is fully configured and ready for use.

Now you are ready to use yours Essential Tools!
